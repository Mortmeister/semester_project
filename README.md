# Community Science Museum

Welcome to my repository for the Community Science Museum Website! This project is a responsive, interactive website designed to inform and engage middle school students, families, and teachers about the exciting exhibits and events offered at the museum. A front-end website for a fictional science museum, built with HTML, CSS, and JavaScript.

Live demo: https://community-science-museum-oslo.netlify.app/

## Pages

- Home
- Exhibits (Biology and Medicine, Cosmology, Ecology, Evolution, Robotics and AI)
- Visit
- Events
- Explore (For Kids, For Teachers, For Researchers)
- Support
- Contact

## Tech stack

- HTML5
- CSS

## Getting started

```
git clone git@github.com:Mortmeister/semester_project.git
cd community-science-museum
```

Clone the repo and open on liveserver in your browser.
